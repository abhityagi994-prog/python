# Python Repository

This repository contains structured Python practice work covering core programming fundamentals to advanced concepts.  
The purpose of this repository is to strengthen problem-solving skills and build a solid foundation in Python.

---

## 📌 Topics Covered

- Python Basics (syntax, variables, loops, conditionals)
- Data Structures (lists, tuples, sets, dictionaries)
- Functions and modular programming
- Object-Oriented Programming (OOP)
- File handling
- Exception handling
- Working with modules and packages
- Basic problem-solving exercises

---

## 📁 Project Structure
```
- `scripts/` → Python scripts from practice sessions.
- `notes/` → Weekly notes summarizing concepts learned.
- `README.md` → Project overview.
- `.gitignore` → Ignore unnecessary files.
```

## ▶️ How to Run

From the project root directory:

```bash
python scripts/<filename>.py
```
Example:

```bash
python scripts/demo_file.py
```
